### 受付番号の生成 (自動採番)

- 1–50 の番号が書かれたプラカードを用意する
- ラミネート済みのプラカードは 3 部用意しておく (1 日周期で欠番を埋める)
- 人間が失くすことを想定して, 生成するときに任意の回数飛ばせるようにする
